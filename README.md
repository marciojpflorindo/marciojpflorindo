# Márcio Florindo

Senior Technical Writer focused on documentation systems, AI-assisted knowledge operations, developer experience, and docs-as-code workflows.

I work on the infrastructure around documentation: research workflows, support-signal analysis, review systems, migration tooling, and structured content operations for complex technical products. The writing is the visible output; the deeper work is designing systems that help teams produce, maintain, and retrieve reliable product knowledge.

At Cloudflare, that has included AI-assisted documentation workflows adopted across multiple teams, support-case analysis for enterprise networking products, large-scale documentation migrations, and tooling that reduced research and setup time for recurring documentation work.

## Current Focus

- Documentation systems for complex products, especially networking, cloud infrastructure, security, and developer platforms
- Support analytics and troubleshooting architecture based on real customer friction
- Docs-as-code workflows, GitHub-based review processes, and reusable operational tooling
